elBaladiya.tn est construit avec les technologies modernes suivantes :

  • Backend: Spring Boot (Java 21) avec architecture REST
  • Frontend: Angular 19 avec Material Design
  • Sécurité: JWT (JSON Web Tokens) pour l'authentification
  • Stockage: XML pour la persistance des données
  • Cartographie: Leaflet pour la visualisation géographique

Gouvernance

Le projet elBaladiya.tn est développé dans le cadre d'une initiative de modernisation des services municipaux. La plateforme est conçue pour être évolutive et adaptable aux besoins spécifiques de chaque municipalité.
